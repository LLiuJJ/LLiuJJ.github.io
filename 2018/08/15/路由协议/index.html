路由协议

Posted on 2018-08-15 |

路由控制的定义

  • 为了让数据包能够正确的发送到目标主机，路由器必须在途中进行正确地转发。这种向“正确的方向”转发数据所进行的处理就叫做陆游控制或路由。
  • 静态路由的设置通常是实现设置好路由器和主机并将路由信息固定的一种方法。
  • 动态路由是指让路由协议在运行过程中自动地设置路由控制信息的一种方法。

路由控制范围

  • ICP - Interior Gateway Protocol (内部网关协议)

  • EGP - Exterior Gateway Protocol (外部网关协议)

  • IGP中还可以使用RIP(Routing Information Protocol，路由信息协议)、RIP2、OSPF(Open Shortest Path First, 开放最短路径优先)等众多协议。

  • EGP使用的是BGP(Porder Gateway Protocol，边界网关协议)协议

路由算法

距离向量算法

路由器之间可以呼唤网络的方向及其距离相关信息，并以这些信息制作路由控制表。这种方法在处理上比较简单，不过由于只有距离和方向的信息，所以当网络变得十分复杂的时候，在获得稳定路由信息之前需要消耗一定的时间。

链路状态算法

路由状态算法是路由器在了解网络整体连接状态的基础上生成路由控制表的一种方法。该方法中，每个路由器必须保持同样的信息才能进行正确的路由选择

链路状态算法需要各个路由器从网络代理获取路由信息表，在一个规模巨大的网络结构中，管理和处理信息需要高速的CPU处理能力和大量的内存
